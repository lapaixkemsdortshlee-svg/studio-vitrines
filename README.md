# Studio Vitrines

Sites vitrines de démonstration pour startups (stratégie marketing).

## Structure

- `sites/<nom-client>/` — une démo par dossier, isolée.
- `.claude/` — config Claude Code (hook d'auto-setup + réglages).
- `scripts/session-setup.sh` — installe les skills à chaque session.

## Démarrer une démo

Ouvre une session Claude Code sur ce repo et décris le client + l'objectif.
Le hook charge automatiquement les skills design/marketing et gstack.
